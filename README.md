# Menu Shell Script

##  Descrição
Projeto desenvolvido em **Shell Script (Bash)** com o objetivo de estudar comandos Linux e automação via terminal.  
O script apresenta um menu interativo, onde cada opção executa uma funcionalidade específica relacionada ao sistema operacional.

O projeto está em desenvolvimento contínuo, sendo atualizado conforme novos comandos e melhorias são aprendidos.

---

##  Funcionalidades

O menu contém as seguintes opções:

- **MAC** — Mostra o endereço MAC da máquina
- **IP** — Mostra o IP público
- **LOCAL** — Mostra o IP local
- **WIFI** — Teste de conexão com a internet
- **SENHA** — Altera a senha do usuário
- **ORG** — Organiza arquivos de um diretório
- **GNU** — Mostra a versão do Bash instalada
- **MAN** — Mostra o manual (`man`) de um comando
- **APR** — Mostra o `apropos` de um comando
